## Feature 1 — Information Age News Feed in /digest

**Source:** https://ia.acs.org.au/feed (RSS 2.0)
**Coverage:** ~6–8 articles/week — AI, careers, salary, policy, AU tech industry

### What it does
- Fetches the IA RSS feed at build time (Next.js `fetch` with `revalidate`)
- Merges into the existing `/digest` page alongside the current AI news aggregator
- Each IA article shows a "Information Age" source badge (distinct colour from other sources)
- Filtered to categories relevant to AU IT job seekers: AI, careers, workforce, policy, cybersecurity

### Implementation
1. Add `lib/ia-feed.ts` — RSS fetch + parse using `fast-xml-parser` (already in tree) or built-in XML
2. Filter items: exclude marketing/events noise, keep tech + career articles
3. Merge with existing digest feed, sorted by `pubDate` desc
4. Source badge: `#0369a1` blue with "Information Age" label

### Files
| File | Action |
|------|--------|
| `lib/ia-feed.ts` | Create — fetch + parse IA RSS, return typed `FeedItem[]` |
| `app/digest/page.tsx` | Modify — merge IA items into existing feed render |

### Acceptance Criteria
- [ ] IA articles appear in `/digest` alongside existing sources
- [ ] Each IA item has source badge "Information Age"
- [ ] Feed revalidates every 6 hours (`revalidate: 21600`)
- [ ] If RSS fetch fails, digest still renders (graceful fallback)
- [ ] No duplicate items if same story appears in multiple feeds

## Feature 2 — ACS Digital Pulse Widget in /au-insights Job Market tab

**Source:** ACS Digital Pulse 2025 Annual Report (acs.org.au/digital-pulse)
**Coverage:** Annual — workforce size, salary growth, gender gap, top skills demand, AI adoption

### What it does
- Adds a "Digital Pulse" highlight card at the top of the Job Market tab
- Shows 5–6 headline stats from the most recent Digital Pulse report
- Clear citation with link to full PDF report
- Visually distinct from the chart sections below — styled as a "report summary" card

### Key stats to embed (Digital Pulse 2025)
- AU tech workforce: 935,000 professionals (+4.2% YoY)
- Median tech salary: $130,000 AUD
- Salary growth: +10% YoY (highest in 5 years)
- AI skills premium: +18% salary uplift for AI-skilled workers
- Gender gap: women = 29% of tech workforce
- Top demanded skill: Cloud/DevOps, then AI/ML, then Cybersecurity

### Implementation
1. Add `app/au-insights/data/digital-pulse.ts` — static typed data, updated annually
2. Create `app/au-insights/DigitalPulseCard.tsx` — stat grid + citation footer
3. Mount at top of `JobMarketCharts.tsx` as a separate highlight section

### Files
| File | Action |
|------|--------|
| `app/au-insights/data/digital-pulse.ts` | Create — ACS Digital Pulse 2025 headline stats |
| `app/au-insights/DigitalPulseCard.tsx` | Create — stat highlight card |
| `app/au-insights/JobMarketCharts.tsx` | Modify — mount DigitalPulseCard at top |

### Acceptance Criteria
- [ ] Card renders at top of Job Market tab
- [ ] All 6 headline stats visible with labels and values
- [ ] Citation footer links to acs.org.au/digital-pulse
- [ ] `_year` and `_accessed` metadata fields present in data file
- [ ] Responsive at 375px — no horizontal scroll

## Feature 3 — JSA Shortage Auto-refresh Script

**Source:** https://www.jobsandskills.gov.au/data/skills-priority-list (Excel download)
**Coverage:** Annual (October each year) — shortage status per ANZSCO occupation, national + state

### What it does
- Replaces manually-maintained `app/au-insights/data/job-market.ts` JSA section
- `scripts/fetch-jsa.ts` downloads the JSA Excel, parses ICT rows (Sub-Major 26), outputs JSON
- Adds `_lastUpdated` timestamp badge to the JSA Skills Matrix component
- Running the script keeps the data fresh without touching component code

### Implementation
1. Create `scripts/fetch-jsa.ts`:
   - Downloads JSA Skills Priority List Excel from public URL
   - Parses with `xlsx` package (add as dev dep)
   - Filters ANZSCO Sub-Major Group 26 (ICT Professionals)
   - Outputs `app/au-insights/data/jsa-skills-auto.json` with `_source`, `_lastUpdated` fields
2. Add `"refresh:jsa": "tsx scripts/fetch-jsa.ts"` to `package.json` scripts
3. Update `JobMarketCharts.tsx` JSA section to show `_lastUpdated` badge

### Files
| File | Action |
|------|--------|
| `scripts/fetch-jsa.ts` | Create — download + parse JSA Excel → JSON |
| `app/au-insights/data/jsa-skills-auto.json` | Generated by script |
| `app/au-insights/JobMarketCharts.tsx` | Modify — show _lastUpdated badge on JSA section |
| `package.json` | Modify — add `refresh:jsa` script |

### Acceptance Criteria
- [ ] `npm run refresh:jsa` runs without errors
- [ ] Output JSON contains all ICT ANZSCO codes with shortage status for latest year
- [ ] `_source` and `_lastUpdated` fields present in JSON
- [ ] JSA matrix in Job Market tab shows "Last updated: Oct 2024" badge
- [ ] Script handles network errors gracefully (exit 1 with message)
